# Project 2: API-Powered DCF Valuation Tool üìä

## Business Context

PE Club is considering investing in **Microsoft (MSFT)**. Instead of manually entering financial data from Bloomberg, you'll build an automated DCF that:
- Pulls latest financials from Yahoo Finance
- Projects cash flows with customizable assumptions
- Calculates intrinsic value
- Generates valuation ranges
- Exports professional Excel model

## Model Components

### 1. **Data Acquisition (Automated)**
```python
import yfinance as yf

# Automatically download:
- Income statement (3 years historical)
- Balance sheet (3 years historical)
- Cash flow statement (3 years historical)
- Current stock price
- Shares outstanding
- Market cap
```

### 2. **Financial Analysis**
```
Historical Analysis:
- Revenue CAGR (3-year)
- EBITDA margin trend
- CapEx as % of revenue
- Working capital changes
- FCF conversion rate
```

### 3. **Projections (5 years)**
```
User-Defined Assumptions:
- Revenue growth rate (with fade)
- EBITDA margin expansion
- Tax rate
- CapEx ratio
- NWC investment

Calculated Automatically:
- EBITDA = Revenue √ó Margin
- D&A (from historical %)
- EBIT = EBITDA - D&A
- Tax = EBIT √ó Tax Rate
- NOPAT = EBIT - Tax
- CapEx = Revenue √ó CapEx %
- Œî NWC = Revenue Growth √ó NWC %
- FCF = NOPAT + D&A - CapEx - Œî NWC
```

### 4. **Terminal Value**
```
Two Methods:
1. Perpetuity Growth: TV = FCF_year5 √ó (1+g) / (WACC - g)
2. Exit Multiple: TV = EBITDA_year5 √ó Exit_Multiple
```

### 5. **WACC Calculation**
```
WACC = (E/V √ó Cost of Equity) + (D/V √ó Cost of Debt √ó (1-Tax))

Where:
- Cost of Equity: CAPM = Rf + Œ≤(Rm - Rf)
- Cost of Debt: From financial data
- E/V, D/V: From balance sheet
```

### 6. **Valuation**
```
Enterprise Value = PV(FCFs) + PV(Terminal Value)
Equity Value = EV - Net Debt
Value per Share = Equity Value / Shares Outstanding

Upside/(Downside) = (Intrinsic Value - Current Price) / Current Price
```
